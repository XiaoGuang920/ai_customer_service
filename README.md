## 關於奧客終結者–評論輔助回覆系統

奧客終結者–評論輔助回覆系統是一個用來輔助商家管理網路評論的平台，透過網路爬蟲等等技術將評論抓取到系統中，藉由統計讓商家管理者能對於商家在網路上的評論能夠一目了然，並藉由結合ChatGPT的輔助讓管理者能夠將回覆評論的時間最小化。
